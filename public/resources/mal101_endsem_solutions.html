1 Eigenvalues & Diagonalization

Let A = [[4, 8, -2], [-3, -6, 1], [9, 12, -5]].
(a) Find the eigenvalues of A.
(b) Determine whether A is diagonalizable.
(c) If A is diagonalizable, find an invertible matrix P and a diagonal matrix D such that A = PDP-1.

Solution Hint (a): Eigenvalue Derivation

det(A - λI) = 0

| 4-λ 8 -2 |
| -3 -6-λ 1 | = 0
| 9 12 -5-λ |

Expanding the characteristic equation yields:
-λ³ - 7λ² - 16λ - 12 = 0 ⇒ -(λ + 2)²(λ + 3) = 0

Hence, eigenvalues are λ = -2, -2, -3. 1 Mark

Solution Hint (b): Eigenvectors Investigation

For repeated eigenvalue λ = -2, compute null space of A + 2I:

A + 2I = [[6, 8, -2], [-3, -4, 1], [9, 12, -3]]
Performing Row Reduction (R2 → R2 + 0.5 R1, R3 → R3 - 1.5 R1) yields:
[[6, 8, -2], [0, 0, 0], [0, 0, 0]]

From 6x + 8y - 2z = 0 ⇒ 3x + 4y - z = 0
Eigenvectors can be written as:
v₁ = (1, 0, 3)ᵀ, v₂ = (0, 1, 4)ᵀ. 1 Mark

For eigenvalue λ = -3, compute null space of A + 3I:

A + 3I = [[7, 8, -2], [-3, -3, 1], [9, 12, -2]]
Row reduced echelon: [[1, 2, 0], [0, 3, 1], [0, 0, 0]]
Implies x + 2y = 0 and 3y + z = 0.
For y=1, eigenvector is v₃ = (2, -1, 3)ᵀ. 1 Mark

Solution Hint (c): Diagonalization Matrices

Since we found three linearly independent eigenvectors, A is diagonalizable:

P (Modal Matrix):
[[ 2, 1, 0 ],
[-1, 0, 1 ],
[ 3, 3, 4 ]] [0.5 Mark]
D (Diagonal Matrix):
[[-3, 0, 0 ],
[ 0, -2, 0 ],
[ 0, 0, -2 ]] [0.5 Mark]

2 Gram-Schmidt Process & Projections

(a) Let V = P₂([R]) with the inner product <f, g> = ∫₋₁¹ f(x)g(x) dx. Find an orthonormal basis of V.
(b) Let P₃([R]) with the same inner product. Compute the orthogonal projection of f(x) = x³ on V.

Solution Process (a) - Gram-Schmidt

Step 1: Basis Vector 1 (u₁ = 1)
||u₁||² = ∫₋₁¹ 1 dx = 2 ⇒ ||u₁|| = √2.
e₁ = u₁/||u₁|| = 1/v2. [0.5 Mark]
Step 2: Basis Vector 2 (u₂ = x - <x, e₁>e₁)
<x, e₁> = (1/√2) ∫₋₁¹ x dx = 0.
Hence, u₂ = x. ||u₂||² = ∫₋₁¹ x² dx = 2/3.
e₂ = u₂/||u₂|| = v(3/2) x. [0.5 Mark]
Step 3: Basis Vector 3 (u₃)
u₃ = x² - <x², e₁>e₁ - <x², e₂>e₂
<x², e₁> = 1/√2 * 2/3 = (√2)/3.
<x², e₂> = 0 (integrates odd function x³).
u₃ = x² - (√2/3)*(1/√2) = x² - 1/3.
||u₃||² = ∫₋₁¹ (x² - 1/3)² dx = 8/45.
e₃ = v(45/8) * (x² - 1/3).

Orthonormal basis: { 1/v2, v(3/2)x, v(45/8)(x² - 1/3) }. [1 Mark]

Solution Process (b) - Orthogonal Projection

proj_V(f) = Σ <f, e_i> e_i

<x³, e₁> = (1/√2) ∫₋₁¹ x³ dx = 0
<x³, e₂> = √(3/2) ∫₋₁¹ x⁴ dx = √(3/2) * (2/5)
<x³, e₃> = 0 (odd integrand)

Resulting projection term:
proj_V(x³) = <x³, e₂>e₂ = [√(3/2)*(2/5)] * [√(3/2) x] = 3/5 x. [1 Mark]

3 Linear Adjoints & Uniqueness Theorems

Let T be linear operator on [R]³ defined as T(x,y,z) = (-x+y, 5y, 4x-2y+5z). Find T*, and check self-adjointness.

Standard matrix formulation of T:
A = [[ -1, 1, 0 ],
     [ 0, 5, 0 ],
     [ 4, -2, 5 ]] [1 Mark]

Matrix of T* is transpose Aᵀ:
Aᵀ = [[ -1, 0, 4 ],
      [ 1, 5, -2 ],
      [ 0, 0, 5 ]] [1 Mark]

Hence, adjoint operator formula is:
T*(x,y,z) = (-x + 4z, x + 5y - 2z, 5z) [0.5 Mark]

To test self-adjointness: since A ≠ Aᵀ, T is not self-adjoint. [0.5 Mark]
Picard's Uniqueness Proof

Show whether dy/dx = √y + 1, y(0)=0 has a unique solution on 0 ≤ x,y ≤ 1.

We check the ratio for Lipschitz continuity in the second variable:

|f(x, y₁) - f(x, y₂)| / |y₁ - y₂| = |√y₁ - √y₂| / |y₁ - y₂| = 1 / (√y₁ + √y₂)

This ratio is unbounded as y₁, y₂ approach 0. Therefore, f is not Lipschitz continuous around 0, and uniqueness cannot be concluded from standard Picard's existence theorem. [2 Marks]

4 Laplace Transforms & ODEs

Inverse Laplace

Find the Inverse Laplace of F(s) = (s + 10) / (s² + 8s + 20).

s² + 8s + 20 = (s + 4)² + 4
F(s) = [s + 4] / [(s+4)² + 4] + 6 / [(s+4)² + 4]

Inverse transform using standard shift theorem results:
L⁻¹{ (s+a) / ((s+a)² + b²) } = e⁻ᵃᵗ cos bt
L⁻¹{ b / ((s+a)² + b²) } = e⁻ᵃᵗ sin bt

Hence: f(t) = e⁻⁴ᵗ (cos 2t + 3 sin 2t). [1 Mark]
Laplace ODE Solution

Solve the second-order IVP: y'' + 7y' + 10y = 4t e⁻³ᵗ, given y(0)=0, y'(0)=-1.

Taking Laplace transforms:
(s²Y - sy(0) - y'(0)) + 7(sY - y(0)) + 10Y = 4 / (s+3)²
Substitute initial conditions:
(s²Y + 1) + 7sY + 10Y = 4 / (s+3)²
Y(s) [s² + 7s + 10] = [4 / (s+3)²] - 1 ⇒ s² + 7s + 10 = (s+5)(s+2)

Taking partial fractions and computing inverse Laplace:
Resulting solution:
y(t) = e⁻²ᵗ - (1 + 2t)e⁻³ᵗ. [1 Mark]
